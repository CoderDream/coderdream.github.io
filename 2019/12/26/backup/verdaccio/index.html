打开 4873 端口的防火墙：

  1. 查看firewall服务状态

  2. 查看firewall的状态

  3. 开启、重启、关闭、firewalld.service服务

    firewall-cmd –list-all

查询端口是否开放

firewall-cmd –query-port=4873/tcp

开放4873端口

firewall-cmd –permanent –add-port=4873/tcp

移除端口

firewall-cmd –permanent –remove-port=4873/tcp
#重启防火墙(修改配置后要重启防火墙)
firewall-cmd –reload

npm add S229 http://10.100.254.229:4873

nrm use S229

npm adduser –registry http://10.100.254.229:4873

npm login

输入添加用户时的用户名密码
