命令行方式执行mocha

  • npm 允许在package.json文件里面，使用scripts字段定义脚本命令。

  • 命令行下使用npm run命令，就可以执行这段脚本。

  • 说明：

  1. npm run test，执行所有的test文件；
  2. npm run test:mocha，执行特定的文件test/mapstorage_test.js；
  3. nyc -a –reporter=html，生成覆盖率报告。
