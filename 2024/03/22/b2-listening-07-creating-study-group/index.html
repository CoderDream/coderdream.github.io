Matched Definitions | 匹配定义

  1. an assignment - a piece of work given to someone, normally as part of their studies or job

    • 作业 - 分配给某人的一项工作，通常是作为他们学习或工作的组成部分

  2. a compromise - an agreement in which people in an argument reduce or change what they are asking for in order to agree

    • 妥协 - 一种协议，在争论中双方减少或改变他们的要求以达成一致

  3. go ahead - an informal expression used to give someone permission to start doing something

    • 继续 - 一个非正式的表达，用来给予某人开始做某事的许可

  4. annoyed - irritated, angry

    • 烦恼的 - 恼怒的，生气的

  5. to learn something by heart - to learn something so that you can say it from memory

    • 牢记 - 记住某事，以便你能背诵

  6. a moderator - a manager of a public discussion

    • 调解人 - 公共讨论的管理者

Dina: Can I say something here?

Bea: Yeah. I’ve never been there but …


Ali: So, we ought to decide how long for and how often.


Dina: I read somewhere that you should make the meeting at the same time each week. Like a seminar. That way we’d take it more seriously.


Bea: We may as well make it for this time since we’re all here. Is this time OK?


Dina: Works for me.


Ali: Me too.

Dina: Umm, no.


Bea: Thursday, no? 


Ali: No, that’s on Thursday. 


Chris: Sorry. Forget I said anything.


Ali: Don’t worry about it.


Bea: So everyone agrees that this time is fine? Every week?


Ali: How long should we make it?


Bea: An hour?
