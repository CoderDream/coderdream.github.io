Grammar explanation语法解释

We use comparative adjectives to compare two things or show change. The comparative form depends on the number of syllables in the adjective.我们使用形容词比较级来比较两个事物或显示变化。比较级形式取决于形容词的音节数。

Adjectives with one syllable一音节形容词

To make comparative forms with one-syllable adjectives, we usually add -er:用单音节形容词构成比较级时，我们通常加-er：

old → older老→老
clean → cleaner干净→清洁
slow → slower慢→慢

If an adjective ends in -e, we add -r:如果形容词以 -e 结尾，我们添加 -r：

safe → safer安全→更安全
nice → nicer不错 → 更好

If an adjective ends in a vowel and a consonant, we usually double the consonant:如果形容词以元音和辅音结尾，我们通常将辅音加倍：

big → bigger大→更大
hot → hotter热→更热

Adjectives with two or more syllables有两个或多个音节的形容词

If a two-syllable adjective ends in a consonant and -y, we change -y to -i and add -er:如果一个双音节形容词以辅音和-y结尾，我们将-y改为-i并添加-er：

noisy → noisier吵闹→更吵闹
happy → happier快乐→更快乐
easy → easier容易 → 更容易

We use more to make comparative forms for most other two-syllable adjectives and for all adjectives with three or more syllables:对于大多数其他双音节形容词以及所有具有三个或更多音节的形容词，我们使用 more 来制作比较级形式：

crowded → more crowded拥挤→更加拥挤
stressful → more stressful压力大→压力更大
dangerous → more dangerous危险→更危险

Exception: You can either add -er/-r or use more with some two-syllable adjectives, such as common, cruel, gentle, handsome, likely, narrow, pleasant, polite, simple and stupid.例外：您可以添加 -er/-r 或与一些双音节形容词一起使用 more，例如 common、cruel、gentle、enjoy、likely、narrow、please、police、simple 和 Stupid。

I think life in the countryside is simpler* than in the city.我认为农村的生活比城市简单。
It’s它是more simple*** *to live in the city because everything you need is there.*在城市生活更简单，因为你需要的一切都在那里。

Irregular adjectives不规则形容词

The adjectives good, bad and far have irregular comparative forms:形容词good、bad和far有不规则的比较级形式：

good → better好→更好
bad → worse坏 → 更糟
far → further/farther远 → 更远/更远

Than比

When we want to say which person or thing we are comparing with, we can use than:当我们想要表达我们正在与哪个人或事物进行比较时，我们可以使用than：

Their house is cleaner than* ours.他们的房子比我们的干净。
Traffic is交通是slower*** in the city than* in the countryside.城市比乡村慢。
After the race I was比赛结束后我more tired than*** *Anne.*比安妮还累。

Grammar explanation语法解释

Nouns can be countable or uncountable. Countable nouns can be counted, e.g. an apple, two apples, three apples, etc. Uncountable nouns cannot be counted, e.g. air, rice, water, etc. When you learn a new noun, you should check if it is countable or uncountable and note how it is used in a sentence.名词可以是可数或不可数。可数名词是可以计算的，例如一个苹果、两个苹果、三个苹果等。不可数名词不能计数，例如空气、大米、水等。当你学习一个新名词时，你应该检查它是可数还是不可数，并注意它在句子中的用法。

Countable nouns可数名词

For positive sentences we can use a/an for singular nouns or some for plurals.对于肯定句，我们可以使用 a/an 表示单数名词，或使用 some 表示复数名词。

There’s a man at the door.门口有一个男人。
I have some friends in New York.我在纽约有一些朋友。

For negatives we can use a/an for singular nouns or any for plurals.对于否定词，我们可以使用 a/an 表示单数名词，或使用 any 表示复数名词。

I don’t have a dog.我没有狗。
There aren’t any seats.没有座位了。

Uncountable nouns不可数的名词

Here are some examples of uncountable nouns:以下是不可数名词的一些示例：

bread面包 rice米 coffee咖啡 information信息
money钱 advice建议 luggage行李 furniture家具

We use some with uncountable nouns in positive sentences and any with negatives.我们在肯定句中将 some 与不可数名词一起使用，而 any 与否定句一起使用。

There’s some milk in the fridge.冰箱里还有一些牛奶。
There isn’t any coffee.没有咖啡。

Questions疑问句

In questions we use a/an, any or how many with countable nouns.在疑问句中，我们使用 a/an、any 或 how much 与可数名词一起使用。

*Is there an email address to write to?有可以写信的电子邮件地址吗？
Are there any chairs?这里有椅子吗？
How manychairs are there?这里有多少椅子？*

And we use any or how much with uncountable nouns.我们将any 或how much 与不可数名词一起使用。

*Is there any sugar?有糖吗？
How muchorange juice is there?这里有多少橙汁？*

But when we are offering something or asking for something, we normally use some.但是当我们提供某物或要求某物时，我们通常会使用一些。

Do you want some chocolate?你想要一些巧克力吗？
Can we have some more chairs, please?请给我们多一些椅子，好吗？

We also use some in a question when we think the answer will be ‘yes’.当我们认为答案是“是”时，我们也会在问题中使用 some。

Have you got some new glasses?你有新眼镜吗？

Other expressions of quantity其他数量表达方式

A lot of (or lots of) can be used with both countable and uncountable nouns.a lot of (or much of) 可以与可数名词和不可数名词一起使用。

There are lots of apples on the trees.树上有很多苹果。
There is a lot of snow on the road.路上有很多雪。

Notice that we don’t usually use many or much in positive sentences. We use a lot of instead.请注意，我们通常不会在肯定句中使用 much 或 much。我们用 a lot of 代替。

They have a lot of money.他们有很多钱。

However, in negative sentences we use not many with countable nouns and not much with uncountable nouns.然而，在否定句中，我们对可数名词使用 not many，对不可数名词使用 not much。

There are a lot of carrots but there are*n’t many potatoes.胡萝卜很多，但土豆不多。
There’s lots of juice but there isn’t much water.果汁很多，但水不多。*
